Abstract

A set of 6 online Partial Discharge measurements were given to the Project Team. The task was to build a MATLAB program that analyses the data using a technique called Pulse Sequence Analysis. A program with advanced features was successfully developed and demonstrated:

  • An intuitive GUI based interface
  • Online signal denoising using wavelet packets,
  • Pulse Sequence Analysis algorithm and graph plotting,
  • Feature extraction using competitive neural networks,
  • Computer automated analysis by comparing features between data sets.

The program executed on a standard PC in less than 30 seconds. In the final presentation, the team were able to characterise the Partial Discharge using the graphs produced in Pulse Sequence Analysis. This was confirmed using the automated feature extraction and comparison. All source code is provided in the repository.
